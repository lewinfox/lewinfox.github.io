A summary of my open-source projects.

### `levitate`

[`levitate`](https://lewinfox.github.io/levitate) is a fuzzy string matching package for R, based on
the `fuzzywuzzy` Python package.


### `icecream`

[`icecream`](https://lewinfox.github.io/icecream) provides user-friendly print debugging in R.


### `rmsfact`

[`rmsfact`](http://pypi.org/project/rmsfact) is a pointless Python package that tells you lies about
Richard M. Stallman.

### `foodwebr`

[`foodwebr`](https://github.com/lewinfox/foodwebr) helps you draw function dependency graphs in R.
